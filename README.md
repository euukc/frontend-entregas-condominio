# Sistema de Gestão de Entregas para Condomínios

## Projeto Integrador – Análise e Desenvolvimento de Sistemas (Senac)
Este projeto é uma solução desenvolvida para otimizar o fluxo de recebimento e retirada de encomendas em condomínios residenciais. A plataforma permite que porteiros registrem entradas e moradores confirmem retiradas, centralizando o controle e evitando extravios.

## Tecnologias
- HTML5
- CSS3 (TailwindCSS via CDN)
- JavaScript (Vanilla)

## Como executar
Abra o arquivo `index.html` no navegador.

## Hospedagem no GitHub Pages
Este projeto está configurado para ser hospedado no GitHub Pages. Para publicar:

1. Faça push deste repositório para o GitHub.
2. Vá para Settings > Pages.
3. Selecione a branch `main` e a pasta raiz `/`.
4. Salve e aguarde a publicação.

O site estará disponível em `https://<username>.github.io/<repo-name>/`.

## Funcionalidades
- Página inicial com informações do sistema
- Login e registro de usuários (simulado)
- Dashboard para moradores: visualizar entregas
- Dashboard para porteiros: registrar entregas e visualizar pendentes
